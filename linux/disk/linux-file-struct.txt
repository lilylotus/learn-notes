一. Linux 文件的组成(三部分) : 文件名， inode， block(真正存放数据)
    inode(索引节点) 存储文件的元信息 : 文件创建者，创建日期，文件大小
    inode 的内容 :
        文件的字节数
        文件的拥有者 (User ID)
        文件的所属组 (Group ID)
        文件的读、写、执行权限
        文件的时间戳 : 
            ctime (文件的权限、拥有者、所属的组等改变时的时间)
            mtime (文件内容被修改的最后时间)
            atime(文件中的数据最后被访问的时间)
        连接数 : 多少文件名指向这个inode
        文件block的位置
注意 : inode 大小是有限制的，也会消耗硬盘空间
每个文件最少有一个inode号，操作系统操作的是inode

1. 查看文件状态
    > stat [文件]
2. 查看 inode 的数量
    > df -i 
3. 查看inode的号码
    > ls -i [文件/目录]
4. ；列出文件的时间戳
    > ls -lc [文件] (ctime)
    > ls -lu [文件] (atime)
    > ls -l  [文件] (mtime)
