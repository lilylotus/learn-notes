1. WHERE 标签 <where></where>
where 元素只会在至少有一个子元素的条件返回 SQL 子句的情况下才去插入 WHERE子句。
而且，若语句的开头为 AND 或 OR ，where 元素也会将它们去除

也可以用 trim 标签来处理
<trim prefix="WHERE" prefixOverrides="AND |OR ">
  ... 
</trim>
prefix：前缀，prefixoverride：去掉第一个 and 或者是 or 
suffixoverride：去掉最后标记的字符（就像是上面的and一样）
suffix：后缀

2. SET <set></set>
set 标记自动把最后一个逗号给去掉
