打包时候不测试：
mvn install -Dmaven.test.skip=true 或者 mvn install -DskipTests

自己 install 包到本地仓库：
mvn install:install-file -DgroupId=ojdbc -DartifactId=ojdbc6 -Dversion=11.2.0.4 -Dpackaging=jar -Dfile=ojdbc6.jar
